Taro(React Native)框架采坑笔记

2018-08-03

现在已经是凌晨1点，起因是下午使用了Taro一个组件，无法达到预定回调结果，刚刚解决，结果是因为马虎问题浪费了小半天的时间，特此写此篇文章，记录并收集使用Taro框架中遇到的问题，以及解决问题的思路。

遇到问题时，解决问题的思路如下：

  1. 大小写！大小写！大小写！ (重要的问题说三遍)
  2. 检查所使用的组件为原生组件或是高级封装，如果为高级封装则检查文档中与原生组件的不同之处
  3. 百度、Google所报的错误，提取简要关键信息，去除杂项(项目路径、个人姓名等)
  4. 到项目GitHub中提交issues，注意简要说明问题，给出问题复现方法，以及问题代码，既然是想让作者帮你解决问题，一定要让作者明白你所阐述的问题所发生的原因

遇到问题时随时补充
