Ubuntu安装Node.js环境

2018-06-20

Quick Start

Ubuntu系统环境

只检查不更新已安装的软件包

$ apt update

更新已安装的软件包

$ apt upgrade

Node.js安装

安装Node.js软件包

$ apt install nodejs

安装Node的编译、运行所需的环境变量

$ apt install nodejs-legacy

安装包管理工具

$ apt install npm

安装Node更新所需的工具”n”

$ npm install n -g

更新node版本

$ n stable

安装MySQL环境

安装MySQL服务

$ apt install mysql-server

安装MySQL客户端

$ apt install mysql-client

配置安全向导

$ mysql_secure_installation
